§ 14.B: The cat Command

The cat (concatenate) command is exceptionally versatile. Its most common use is to view a file:

    > cd ~

    > cat .forward

If you have done as told and created a .forward file when you worked through the email section, Unix should display your email address.

Because the entire file will flash on the screen with cat, it is good only for short files. The view and more commands are better for files with more than a few lines.

Next we will see how, in combination with the special symbols <, >, >> and |, cat can be used for other purposes.
