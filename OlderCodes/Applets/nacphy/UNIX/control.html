§ 5.L:  Controlling Access to Files and Directories

One of the advantages of Unix is that it allows many users to work from the same file system and even to access and modify shared files. Unfortunately, this can also be a disadvantage if someone reads or modifies your files without your permission. After we discuss how to determine and understand file permissions, we will see how you can protect yourself by setting file permissions.
