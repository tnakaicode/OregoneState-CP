§ 12.A.1: Mail Forwarding

Your .forward (pronounced "dot-forward") file can be created with your favorite editor. Use the editor to create a file named .forward, and place in it a single line containing your email address:

    yourusername@ucs.orst.edu
When done editing and saving the file, check it:
    > more .forward
If you have accounts on a number of machines, it makes good sense to have the same .forward file on all of them but one. You can do this by creating the same .forward file on each machine, or ftping it to each.
