§ 18: Maple

A very useful program on the Physics Department cluster is the problem solving environment package Maple. Maple is useful for solving equations both analytically and numerically and for visualizing results. To activate the system, you issue the commands

maple

    start Maple in telnet or Webterm
xmaple

    start Maple in X window

The X window is nicer because it allows the use of the mouse and the pretty display of mathematical symbols. Whichever Maple you use, the commands to solve equations are the same.
