Elm

Elm is a little easier to use that Pine, but the editor it uses to create the message is in vi, and for some people, that's enough to convince them not to use elm. For this reason and the fact that the physics cluster doesn't have elm, we're not going to go through an example. If you want to try it out, just type 'elm' at the ucs prompt. It will ask to create a mail directory and file for elm, you need these to continue mailing in elm, so type 'y' at each respective prompt. If you would like some further information on elm, then you can type 'man elm' at the UCS prompt, or you can click here.
