§ 9.D:  Cancelling A Print Job

Let's say that without thinking (an understatement) you accidentally sent the Encyclopedia Brittanica to the printer. This is a case where you need to cancel a job in the print queue or already printing.

Use the line printer remove or cancel commands followed by the job number to cancel a print job:

    > lprm 601
    > cancel 601
Sometimes Unix tells you the job number when you submit your job. You can always get the job number from the print queue listing:
    > lpq 
     daphy: Fri Jul 26 14:47:49 1996: Fri Jul 26 14:46:58 1996:
Attempting to print dfA601daphy Rank Pri Owner Job Files Total Size
active 0 rubin 601 cp05.ps 375122 bytes

Queue   Dev   Status    Job Files              User         PP %   Blks  Cp Rnk
------- ----- --------- --- ------------------ ---------- ---- -- ----- --- ---
ps2     @goop READY    
ps2: Fri Jul 26 14:46:58 1996: Attempting to print dfA601daphy
ps2     ps2   RUNNING     0 601                 0             0  0     1  1   0
We see that the job number is 601.

Because we do not have to cancel the Encyclopedia Brittanica, we will go on to Smart Printing.
