§ 4: Courtesy, Ethics, and Security

A Unix workstation is often shared among a number of users, with each user capable of spawning a number of simultaneous tasks for the computer. A cluster of workstations may involve even more sharing. Under these conditions, it is only reasonable for you to follow some basic courtesies for the benefit of all. In addition, there are ethical, legal, and practical considerations when using an open system owned by a government agency. And finally, security measures are necessary to safeguard both the computers and the work you have in them.

Don't be too concerned if, when first reading these sections, you do not understand all the commands and procedures. In that case a second reading may be helpful, especially after you have done some more work with Unix. If you still have questions, ask support. We recommend that you refer to these rules and procedures frequently.
