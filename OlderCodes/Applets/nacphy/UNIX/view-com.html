§ 6.C:  The View Command

The view command is a read-only version of the vi editor (we'll get to vi soon). The view command is comforting, it allows you to look at a file without the chance of messing it up. Try:

    > view ~/README.first
Unix should respond by printing the file on the screen. To move through the file, use some of the standard vi commands:
    [return] key scroll down one line at a time
    ^f (ctrl + f) move forward one page
    ^b move back one page
    :q quit view
    More commands same as vi

Now that you know how to view files, it's time to learn how to edit them.
